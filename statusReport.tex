\title{Modulo7 Write Up: Request for ground truth data from MIREX 2005}

\maketitle

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\noindent Modulo7 is a new framework currently under construction to measure similarities across songs of different sources, extract features and observe music theoretic criteria satisfaction for disparate song sources. Modulo7 outputs quantitative similarities between two music sources as one of its components.\\\\
In order to facilitate evaluation of these measures, ground truth information is needed for song similarities. One such ground truth measure was established in the MIREX competition (Music Information Retrieval Evaluation eXchange ). While most of the datasets is proprietary hosted in UIUC, the 2005 version was downloadable for the task of Symbolic Melodic similarity \href{http://www.music-ir.org/mirex/wiki/2005:Symbolic_Melodic_Similarity}{Link to dataset description}. \\\\
The data that was originally hosted is taken down. Is it possible to get a copy of the dataset for this research?
